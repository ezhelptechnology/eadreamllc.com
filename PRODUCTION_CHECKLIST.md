## üÜò ROLLBACK PLAN

**If deployment fails:**

1. Identify error in Vercel logs
2. If database-related: Revert migration
   ```bash
   npx prisma migrate resolve --rolled-back <migration_name>
   ```
3. If code-related: Revert to previous deployment in Vercel dashboard
4. If OAuth-related: Re-authorize at `/api/auth/google`
